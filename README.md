## ğŸš€ Getting Started

### Prerequisites
- Node.js 18+ 
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/asamarsal/INJHyperBoard.git

# Navigate to project directory

# Install dependencies
npm install

# Run development server
npm run dev
```
